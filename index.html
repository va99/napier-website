Napier

Configure MCP directly from prompts

Napier is a special MCP server that installs other MCP servers for you. Once it's set up, you can ask Claude to install MCP servers hosted on npm or PyPi — no manual installs needed. It requires npx for Node-based servers and uv for Python-based ones.

How to set it up

  1. Install Node.js if you don’t have it.
  2. Edit your Claude config file and add the following:
{
  "mcpServers": {
    "napier-mcp": {
      "command": "npx",
      "args": [
        "napier-mcp"
      ]
    }
  }
}

File location:

  • macOS: ~/Library/Application Support/Claude/claude_desktop_config.json
  • Windows: C:\Users\YOU\AppData\Roaming\Claude\claude_desktop_config.json

What you can say to Claude

  • Hi Claude, please install the MCP server at /Users/bojack/code/whatsapp-mcp, I'm too lazy to do it myself.
  • Hey Claude, install the @notion-mcp package as an MCP server. Use ['/Users/bojack/Desktop'] for the arguments.
